# Bay Area Events Finder 🎉

AI-powered discovery of social events for people in their 20s across the Bay Area.

## Features

✅ Automatic weekly email digest every Wednesday at 9 AM  
✅ Events within 2 hours of Mountain View  
✅ Organized by region (SF, South Bay, East Bay, Peninsula)  
✅ **COMPLETELY FREE** - Uses Hugging Face (no API costs!)
✅ Free deployment (GitHub Actions + Vercel + Supabase)

## Tech Stack

- **API**: Hugging Face (FREE!)
- **Email**: SendGrid (free tier)
- **Database**: PostgreSQL (Supabase)
- **Automation**: GitHub Actions
- **Hosting**: GitHub + Vercel
